# Mellow

A soothing dark color scheme for neovim.

## Installation

Install the theme with your preferred package manager:

```lua
--packer
use 'ivanjermakov/mellow.nvim'

--lazy
{ "ivanjermakov/mellow.nvim" }
```

## Usage

Enable the colorscheme:

```lua
-- Lua
vim.cmd [[colorscheme mellow]]
```
